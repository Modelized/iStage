Help

Quick start and answers in one page.

Setup

  1. Install KLCK from Google Play.
  2. Go to Releases and download the latest .klck.
  3. Open KLCK → import the preset → apply.

Tips & Answers

• Widgets not showing → Check KLCK permissions (overlay, notifications).
• Performance → Try lowering blur radius or animation scale.
• Migration (18 → 26) → Re-apply widgets to refresh styles.
